## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation
1. Download or clone the files to your local directory
2. Open `index.html` in your web browser
3. For development, use a local server (recommended)

### Using a Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## 🛠️ Customization

### Adding Real Images
Replace the placeholder gallery items in `index.html` with actual clinic photos:

```html
<div class="gallery-item">
    <img src="path/to/your/image.jpg" alt="Clinic Reception">
</div>
```

### Updating Contact Information
Edit the contact details in `index.html` to match your current information.

### Modifying Services
Update the services section in `index.html` to reflect your current offerings.

### Changing Colors
Modify the CSS variables in `styles.css` to match your brand colors:

```css
:root {
    --primary-color: #2c5aa0;
    --secondary-color: #4CAF50;
    --accent-color: #667eea;
}
```
